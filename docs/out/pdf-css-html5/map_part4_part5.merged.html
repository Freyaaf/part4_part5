4 文献导入

4.1 新建文献库

文献库的定义

Zotero 文献库用于组织和管理学术资料。

Zotero 文献库是一个强大的工具，允许用户根据需要创建和管理分类，以便高效存储和检索。

新建 Zotero 文献库并自定义存储位置

  1. 点击屏幕左上角文件菜单，选择新建分类。
  2. 在弹出的名称对话框中输入文献库名称，点击创建分类。
  3. 右键选中文献库，可以新建子分类、重命名文类、删除分类。此处以新建子分类为例。键入子分类名称并点击创建分类。
  4. 快捷方式：直接点击 Zotero 界面左上角的新建分类图标。
  5. 选择屏幕左上角Zotero菜单，点击设置。
  6. 在弹出的设置对话框中，选择高级，点击选择按钮设置自定义存储路径。

4.2 导入文献

4.2.1 从中国知网导入文献条目

从中国知网导入文献条目

Zotero 提供插件支持直接从中国知网（CNKI）导入文献条目，便于高效获取学术资源。

如何从知网导入文献条目

  1. 在中国知网找到所需文献。
  2. 打开“引用”界面，右键点击并选择“Save to Zotero (CNKI)”。
  3. 选择目标文献库保存条目。

4.2.2 通过文献标识符导入题录

通过文献标识符导入题录

文献标识符（如 DOI 或 ISBN）是快速导入条目的关键信息。

如何使用文献标识符导入条目

  1. 点击 Zotero 工具栏的标识符导入图标。
  2. 输入标识符，例如：doi: 10.1007/s43545-021-00273-9，然后回车。

无法找到指定标识符

Condition

Zotero 提示找不到标识符。

Cause

网络连接中断或 VPN 干扰。

Remedy

用户

  1. 确保网络连接正常。
  2. 关闭 VPN。

4.2.3 本地导入中文文献

本地导入中文文献

本地文献导入可手动添加文件到 Zotero 文献库，适合管理非线上资源。

如何从本地导入中文文献

  1. 点击“添加文件”图标。
  2. 从本地选择所需文献。
    文件已成功导入 Zotero。
  3. 右键选中文献，依次点击“知网助手”，“抓取文献元数据”。
    成功抓取文献的元数据并添加至 Zotero。
  4. 右键点击条目，选择“添加到分类”。
    文献已成功添加到目标分类。

文献库中出现重复条目

Condition

Zotero 文献库中出现重复条目。

Cause

文献重复导入。

Remedy

用户

  1. 打开“重复条目”文献库。
  2. 选中其中一条，重复文献将自动被选中。
  3. 点击右上角的“合并条目”完成操作。

5 笔记管理

5.1 添加笔记

5.1.1 高亮

高亮

高亮文本是快速标记文献重点的常用方法，便于后续整理。

在 PDF 文档中添加高亮或下划线

  1. 打开 PDF 文档。
  2. 选中目标句子，点击左侧图标为其高亮，或点击右侧图标为其添加下划线。
    选中的句子已成功高亮或添加下划线。

5.1.2 添加注释

添加注释

注释功能支持多彩标注，帮助高效记录阅读感想或学术见解。

在 PDF 文档中添加注释

  1. 在界面上部选择“添加笔记”图标。
  2. 点击颜色块选择注释颜色。
  3. 单击目标位置，键入笔记内容。
  4. 单击任意空白处以关闭对话框。

5.1.3 删除注释

删除注释

删除不再需要的标注，保持文档整洁。

如何删除注释

  1. 选中注释图标。
  2. 点击右侧“三点菜单”，选择“删除”。
  3. 或在选中注释图标后按键盘 ⌫ 删除键。

5.1.4 通过注释添加笔记

通过注释添加笔记

自动生成基于注释的笔记，并支持格式自定义。

如何通过注释添加笔记

  1. 打开 PDF 文件，右键点击，选择“通过注释添加笔记”。
  2. 自定义笔记内容格式。

5.1.5 添加独立笔记

添加独立笔记

独立笔记适合记录与具体文献条目无关的内容，便于学术管理。

如何添加独立笔记

点击 Zotero 工具栏上的“新建独立笔记”图标。
成功创建一个新的独立笔记。

独立笔记

独立笔记和子笔记在本质上是一样的，只不过独立笔记不属于任何一个条目，独立笔记显示时和条目是一个级别。

5.2 管理笔记

5.2.1 关联文献

关联文献

笔记间可建立条目关联，快速定位相关文献。

如何关联文献

  1. 点击“关联文献”栏中的“+”按钮。
  2. 确定关联文献，点击“选择”。
  3. 添加关联条目后，单击关联的条目就会跳转到该关联条目所在的位置。

关联逻辑

关联关系是双向的，但不具备传递性即条目A与B关联，Zotero会自动让B关联A；若A关联B，B关联C，A并不会自动关联C。。
